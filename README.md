# bevy_sprite_sheet
Create a sprite sheet from an image and [aseprite](https://www.aseprite.org/) data for the bevy ecosystem.

This crate was only ever used in my [pacman](https://github.com/Warhorst/pacman) and is now included directly there.

## Versions
What version of this library supports which bevy version?

| bevy_sprite_sheet version | bevy version |
|---------------------------|--------------|
| 1.0.0                     | 0.16         |
| 1.1.0                     | 0.17         |
| 1.2.0                     | 0.18         |
